Aquarium
A framework for semi-formally representing wetlab protocols and methods as executable computer code. Protocols are written in Plankton, interpreted step-by-step for lab technicians, with every sample and data point tracked for 99% reproducible methods.
Wetlab Protocols LIMS
Software
gro
A language for programming the behavior of cells in growing microcolonies. The simulator models cell growth, division, noise, and diffusing molecular signals. Used to prototype multicell behaviors and verify that local rules produce desired global results.
Synthetic Biology Simulation Language
Software
Icarus
A bare-metal x86 operating system that boots directly on 64-bit UEFI systems. Includes a built-in BASIC interpreter, graphics rendering, disk storage via AHCI, a text editor, and sound — evoking early 1980s home computers.
OS UEFI C BASIC
Software
YASE
A modular digital audio synthesis and processing library designed to teach digital audio synthesis. Use it to make sounds, sequences, and effects. Connect MIDI devices and sound sources.
Audio Synthesis C++
